# Farrouj POS

Lebanese Restaurant Point of Sale System

## Installation

1. Install Python 3.11 or newer
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Run:
   ```
   python main.py
   ```

## Build .exe

```
pip install pyinstaller
python build.py
```

- Username: admin
